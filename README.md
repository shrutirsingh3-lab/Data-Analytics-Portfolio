# 🛒 E-Commerce Analytics SQL Project
## 🔥 Project Highlights
- End-to-end SQL project on real-world dataset
- Covers basic to advanced SQL queries
- Includes business insights and analysis
## 📌 Objective
Analyze customer behavior, sales trends, and product performance using SQL.

---

## 📂 Dataset
Brazilian E-Commerce Dataset (Olist)

---

## 🧠 Key Business Questions Solved

### 🔹 Basic Analysis
- Total orders and revenue
- Customer count
- Monthly order trends

### 🔹 Intermediate Analysis
- Top customers by spending
- Most popular product categories
- Average order value

### 🔹 Advanced Analysis
- Customer ranking using window functions
- Running revenue trends
- Top products in each category

### 🔹 Pro-Level Insights
- Customer retention & churn analysis
- Revenue contribution by category

---

## 🛠️ Skills Demonstrated
- SQL Joins (INNER JOIN)
- Aggregations (SUM, COUNT, AVG)
- GROUP BY & HAVING
- Subqueries
- Window Functions (RANK, DENSE_RANK)
- Business Data Analysis

---

## ⚙️ Tools Used
- MySQL
- SQL

---

## 📈 Key Insights
- Top 10% customers contribute majority of revenue
- Certain categories dominate sales
- High number of one-time customers (churn)
